# MASH AI – Multi-Agent System for Medical Diagnosis

MASH AI is a multi-agent AI system that assists doctors in diagnosing medical conditions using domain-specialized agents and Retrieval-Augmented Generation (RAG).

## Features

- Multi-agent medical reasoning
- Domain-specific knowledge bases
- RAG with multiple medical literature
- AI orchestration layer
- Confidence scoring
- FastAPI backend

## Architecture

Doctor → React UI → FastAPI → Orchestrator → Specialist Agents → Vector DB → Gemini LLM
